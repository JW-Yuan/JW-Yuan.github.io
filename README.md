# JW-Yuan.github.io

个人主页网站，包含以下栏目：
- 首页：个人信息介绍
- GitHub 项目：展示个人项目
- 数据集介绍：分享数据集资源
- 论文阅读心得：记录论文阅读笔记

## 使用说明

1. 编辑 `index.html` 文件，更新个人信息、项目、数据集和论文内容
2. 修改 `assets/css/style.css` 可以自定义样式和颜色
3. 将更改推送到 GitHub，GitHub Pages 会自动部署

## 自定义内容

### 更新个人信息
在 `index.html` 的 `#home` 部分修改个人信息。

### 添加项目
在 `index.html` 的 `#projects` 部分添加新的项目卡片。

### 添加数据集
在 `index.html` 的 `#datasets` 部分添加新的数据集卡片。

### 添加论文心得
在 `index.html` 的 `#papers` 部分添加新的论文卡片。
